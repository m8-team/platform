# @m8/module-sdk

Contract package for M8 Platform UI modules.

The SDK contains only generic contracts and helpers. It does not hardcode concrete modules, scopes, mount points, slots, permissions or capabilities.

## Install inside monorepo

```bash
pnpm add @m8/module-sdk --workspace
```

## Build

```bash
pnpm build
```

## Core rule

Shell code contains contracts. Backend registry contains concrete modules, scopes, mount points and slots.
