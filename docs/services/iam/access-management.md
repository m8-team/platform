# Управление доступом

Раздел `IAM` использует клиентское приложение, проект и user pool как основные
границы выполнения операций аутентификации.

При обработке запроса платформа определяет вызывающего клиента, проверяет его
право запускать authentication workflow и применяет ограничения user pool,
проекта и выбранного способа аутентификации.

## Что включает управление доступом

- аутентификацию вызывающей стороны;
- проверку права клиента запускать операции аутентификации;
- проверку принадлежности клиента к разрешенному проекту и user pool;
- применение политик assurance level и challenge selection;
- контроль идемпотентности повторных запросов запуска аутентификации.

Подробный состав REST-методов и параметры запросов приведены в разделе `Справочник API`.
