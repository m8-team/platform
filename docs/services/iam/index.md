# IAM

Раздел `IAM` описывает аутентификацию пользователей и сервисных клиентов в платформе.

Здесь собраны материалы по запуску authentication workflow, передаче subject hints,
контексту аутентификации и REST API для создания операций аутентификации.

Материалы по модели доступа собраны на странице
[Управление доступом](./access-management.md).
