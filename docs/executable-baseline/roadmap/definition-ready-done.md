# Definition of Ready

[Executable Baseline](../index.md) | [Roadmap Baseline](index.md)

- requirement APPROVED and assigned to release;
- owner context and data owner defined;
- acceptance criteria testable;
- API/event/error contracts linked;
- threat and quality impact assessed;
- Feature/Task Prompts validated.

# Definition of Done

- implementation and migrations merged;
- unit, integration, contract, acceptance and security tests pass;
- audit, metrics, logs and traces present;
- compatibility and architecture gates pass;
- runbook/dashboard updated;
- release evidence linked to requirement.
